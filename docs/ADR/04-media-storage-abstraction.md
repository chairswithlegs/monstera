# ADR 04 — Media Storage Abstraction

> **Status:** Draft v0.1 — Feb 24, 2026  
> **Addresses:** `design-prompts/04-media-storage-abstraction.md`

---

## Design Decisions

| Question | Decision |
|----------|----------|
| Storage key ID | **nanoid** (`github.com/jaevor/go-nanoid`) — short, URL-safe, no time-sortability needed |
| Storage key format | `media/{year}/{month}/{day}/{nanoid}{ext}` — date-sharded for filesystem and S3 listing |
| Upload buffering | Buffer entire body in memory (≤ `MEDIA_MAX_BYTES`, default 10 MB) — simplifies content detection, blurhash, and dimension extraction |
| Blurhash | Synchronous for images in Phase 1; skipped for video/audio (Phase 2) |
| S3 endpoint override | `s3.Options.BaseEndpoint` + `UsePathStyle: true` — the modern AWS SDK v2 approach |
| S3 URL strategy | CDN base URL if `MEDIA_CDN_BASE` is set; otherwise presigned `GetObject` (1-hour TTL) |
| Partial upload rollback | Best-effort `Delete` on storage key when DB insert fails; documented as non-atomic |
| Local file write safety | Write to `.tmp` file then `os.Rename` — prevents serving incomplete files |

## 1. `internal/media/store.go`

```go
package media

import (
	"context"
	"errors"
	"fmt"
	"io"
	"mime"
	"path/filepath"
	"time"

	"github.com/jaevor/go-nanoid"

	"github.com/yourorg/monstera-fed/internal/config"
)

// ErrNotFound is returned by Get when the requested storage key does not exist.
var ErrNotFound = errors.New("media: object not found")

// MediaStore is the pluggable storage backend for user-uploaded media.
// Implementations must be safe for concurrent use.
//
// Keys are opaque strings generated by StorageKey. Callers must treat them as
// opaque — do not parse or construct keys outside this package.
type MediaStore interface {
	// Put stores the content addressed by key. If a value already exists at
	// key it is overwritten (same key always means the same content).
	Put(ctx context.Context, key string, r io.Reader, contentType string) error

	// Get returns a streaming reader for the stored object.
	// The caller is responsible for closing the returned ReadCloser.
	// Returns ErrNotFound if the key does not exist.
	Get(ctx context.Context, key string) (io.ReadCloser, error)

	// Delete removes the object at key. A no-op if the key does not exist.
	Delete(ctx context.Context, key string) error

	// URL returns the public URL at which the object can be fetched.
	// For the local driver this is a path under MEDIA_BASE_URL.
	// For S3 without a CDN this is a presigned GetObject URL.
	URL(ctx context.Context, key string) (string, error)
}

// Config holds the fields the media factory needs.
// Constructed by serve.go from *config.Config.
type Config struct {
	Driver      string // "local" | "s3"
	LocalPath   string // required when Driver == "local"
	BaseURL     string // required — public base URL for local and S3 CDN
	S3Bucket    string // required when Driver == "s3"
	S3Region    string // required when Driver == "s3"
	S3Endpoint  string // optional: endpoint override for MinIO / R2 / B2
	CDNBase     string // optional: CDN prefix; if set, URL() returns CDN URL instead of presigned
	MaxBytes    int64  // maximum upload size in bytes (from MEDIA_MAX_BYTES)
}

// New returns the MediaStore implementation selected by cfg.Driver.
func New(cfg Config) (MediaStore, error) {
	switch cfg.Driver {
	case "local", "":
		if cfg.LocalPath == "" {
			return nil, fmt.Errorf("media: MEDIA_LOCAL_PATH is required when MEDIA_DRIVER=local")
		}
		return local.New(cfg.LocalPath, cfg.BaseURL)
	case "s3":
		if cfg.S3Bucket == "" || cfg.S3Region == "" {
			return nil, fmt.Errorf("media: MEDIA_S3_BUCKET and MEDIA_S3_REGION are required when MEDIA_DRIVER=s3")
		}
		return s3store.New(s3store.Config{
			Bucket:   cfg.S3Bucket,
			Region:   cfg.S3Region,
			Endpoint: cfg.S3Endpoint,
			CDNBase:  cfg.CDNBase,
		})
	default:
		return nil, fmt.Errorf("media: unknown driver %q (valid: local, s3)", cfg.Driver)
	}
}

// nanoidGen is a package-level nanoid generator (URL-safe alphabet, 21 chars).
// Initialised once; safe for concurrent use.
var nanoidGen func() string

func init() {
	var err error
	nanoidGen, err = gonanoid.Standard(21)
	if err != nil {
		panic("media: failed to initialise nanoid generator: " + err.Error())
	}
}

// StorageKey generates a date-sharded, collision-resistant storage key for
// an uploaded file.
//
// Format: media/{year}/{month}/{day}/{nanoid}{ext}
// Example: media/2026/02/24/V1StGXR8_Z5jdHi6B-myT.jpg
//
// Design rationale:
//   - Date prefix shards objects into daily "directories", keeping S3 listing
//     and local filesystem directory scans manageable over time.
//   - nanoid provides 126 bits of entropy — collisions are astronomically unlikely
//     without coordinating across replicas.
//   - The original filename is NOT included in the key: it is user-controlled,
//     may contain path traversal characters, and leaks metadata. The extension is
//     derived from the validated content type, not from the filename.
//   - accountID is not embedded in the key (it is stored in the DB row) to avoid
//     leaking account identity through media URLs.
func StorageKey(contentType string) string {
	now := time.Now().UTC()
	ext := extensionForContentType(contentType)
	id := nanoidGen()
	return fmt.Sprintf("media/%04d/%02d/%02d/%s%s",
		now.Year(), now.Month(), now.Day(), id, ext)
}

// extensionForContentType returns the canonical file extension for the given
// MIME type. Falls back to empty string for unknown types.
func extensionForContentType(ct string) string {
	exts, _ := mime.ExtensionsByType(ct)
	if len(exts) > 0 {
		// mime.ExtensionsByType returns exts in alphabetical order; prefer
		// well-known extensions explicitly.
		preferred := map[string]string{
			"image/jpeg": ".jpg",
			"image/png":  ".png",
			"image/gif":  ".gif",
			"image/webp": ".webp",
			"video/mp4":  ".mp4",
			"video/webm": ".webm",
			"audio/mpeg": ".mp3",
			"audio/ogg":  ".ogg",
		}
		if ext, ok := preferred[ct]; ok {
			return ext
		}
		return exts[0]
	}
	return ""
}

// AllowedContentTypes is the set of MIME types accepted at upload time.
var AllowedContentTypes = map[string]string{
	"image/jpeg": "image",
	"image/png":  "image",
	"image/gif":  "gifv",   // Mastodon treats animated GIF as 'gifv'
	"image/webp": "image",
	"video/mp4":  "video",
	"video/webm": "video",
	"audio/mpeg": "audio",
	"audio/ogg":  "audio",
}
```

## 2. `internal/media/local/local.go`

```go
// Package local provides a filesystem-backed MediaStore implementation.
// Files are stored under a configurable base directory, mirroring the
// storage key structure. The Go HTTP server serves them at /system/...
//
// Not suitable for multi-replica deployments unless the base directory is
// on shared storage (NFS, EFS). Use the s3 driver for multi-replica setups.
package local

import (
	"context"
	"crypto/md5"
	"fmt"
	"io"
	"net/http"
	"os"
	"path/filepath"
	"strings"

	"github.com/yourorg/monstera-fed/internal/media"
)

// Store is the local filesystem MediaStore implementation.
type Store struct {
	basePath string // absolute path to the media root directory
	baseURL  string // public URL prefix, e.g. "https://social.example.com"
}

// New creates a local Store rooted at basePath.
// basePath must be an absolute path; the directory is created if it does not exist.
func New(basePath, baseURL string) (*Store, error) {
	if err := os.MkdirAll(basePath, 0755); err != nil {
		return nil, fmt.Errorf("media/local: create base directory %q: %w", basePath, err)
	}
	return &Store{
		basePath: filepath.Clean(basePath),
		baseURL:  strings.TrimRight(baseURL, "/"),
	}, nil
}

// Put writes the content of r to the file at key under basePath.
//
// Write safety: data is first written to a sibling .tmp file, then atomically
// renamed to the final path. This prevents other goroutines or the HTTP handler
// from reading a partially-written file.
//
// Concurrent writes to the same key: the rename is atomic on POSIX filesystems,
// so the last writer wins. This is acceptable — the same storage key always
// represents the same content (keys are randomly generated and never reused).
func (s *Store) Put(_ context.Context, key string, r io.Reader, _ string) error {
	dst := filepath.Join(s.basePath, filepath.FromSlash(key))
	if err := os.MkdirAll(filepath.Dir(dst), 0755); err != nil {
		return fmt.Errorf("media/local: mkdir for %q: %w", key, err)
	}

	// Write to a temp file in the same directory to ensure rename is atomic
	// (cross-device rename would fail).
	tmp, err := os.CreateTemp(filepath.Dir(dst), ".upload-*")
	if err != nil {
		return fmt.Errorf("media/local: create temp file: %w", err)
	}
	tmpName := tmp.Name()

	// Clean up temp file on any error path.
	ok := false
	defer func() {
		if !ok {
			tmp.Close()
			os.Remove(tmpName)
		}
	}()

	if _, err := io.Copy(tmp, r); err != nil {
		return fmt.Errorf("media/local: write %q: %w", key, err)
	}
	if err := tmp.Close(); err != nil {
		return fmt.Errorf("media/local: close temp file: %w", err)
	}
	if err := os.Rename(tmpName, dst); err != nil {
		return fmt.Errorf("media/local: rename to %q: %w", key, err)
	}

	ok = true
	return nil
}

// Get opens the file at key and returns it as an io.ReadCloser.
// Returns media.ErrNotFound if the file does not exist.
func (s *Store) Get(_ context.Context, key string) (io.ReadCloser, error) {
	path := filepath.Join(s.basePath, filepath.FromSlash(key))
	f, err := os.Open(path)
	if err != nil {
		if os.IsNotExist(err) {
			return nil, media.ErrNotFound
		}
		return nil, fmt.Errorf("media/local: open %q: %w", key, err)
	}
	return f, nil
}

// Delete removes the file at key. A no-op if the file does not exist.
func (s *Store) Delete(_ context.Context, key string) error {
	path := filepath.Join(s.basePath, filepath.FromSlash(key))
	if err := os.Remove(path); err != nil && !os.IsNotExist(err) {
		return fmt.Errorf("media/local: delete %q: %w", key, err)
	}
	return nil
}

// URL returns the public URL for the given storage key.
// Format: {baseURL}/system/{key}
func (s *Store) URL(_ context.Context, key string) (string, error) {
	return s.baseURL + "/system/" + key, nil
}

// ServeHTTP handles GET /system/{key...} requests for locally-stored media.
// It is only registered when MEDIA_DRIVER=local (see router.go).
//
// Features:
//   - Content-Type set from file extension via mime.TypeByExtension.
//   - Cache-Control: public, max-age=31536000, immutable (1 year — media is immutable once stored).
//   - ETag derived from MD5 of the storage key (deterministic; avoids stat() on every request).
//   - If-None-Match conditional GET: returns 304 when ETag matches.
//
// The handler does NOT use http.ServeFile because that function inspects the
// filesystem for last-modified time and adds a redirect for directory requests —
// neither behaviour is appropriate here. We stream the file directly.
func (s *Store) ServeHTTP(w http.ResponseWriter, r *http.Request) {
	// chi wildcard: r.URL.Path is "/system/{key...}" — strip the "/system/" prefix.
	key := strings.TrimPrefix(r.URL.Path, "/system/")
	if key == "" || strings.Contains(key, "..") {
		http.Error(w, "not found", http.StatusNotFound)
		return
	}

	etag := fmt.Sprintf(`"%x"`, md5.Sum([]byte(key)))

	if r.Header.Get("If-None-Match") == etag {
		w.WriteHeader(http.StatusNotModified)
		return
	}

	rc, err := s.Get(r.Context(), key)
	if err != nil {
		if errors.Is(err, media.ErrNotFound) {
			http.Error(w, "not found", http.StatusNotFound)
			return
		}
		http.Error(w, "internal server error", http.StatusInternalServerError)
		return
	}
	defer rc.Close()

	ext := filepath.Ext(key)
	if ct := mime.TypeByExtension(ext); ct != "" {
		w.Header().Set("Content-Type", ct)
	}
	w.Header().Set("Cache-Control", "public, max-age=31536000, immutable")
	w.Header().Set("ETag", etag)

	io.Copy(w, rc)
}
```

```go
// Package s3store provides an S3-compatible MediaStore implementation using
// the AWS SDK v2. Compatible with AWS S3, MinIO, Cloudflare R2, and Backblaze B2.
package s3store

import (
	"context"
	"errors"
	"fmt"
	"io"
	"time"

	"github.com/aws/aws-sdk-go-v2/aws"
	awsconfig "github.com/aws/aws-sdk-go-v2/config"
	"github.com/aws/aws-sdk-go-v2/credentials"
	"github.com/aws/aws-sdk-go-v2/service/s3"
	s3types "github.com/aws/aws-sdk-go-v2/service/s3/types"

	"github.com/yourorg/monstera-fed/internal/media"
)

// Config holds S3-specific configuration.
type Config struct {
	Bucket   string
	Region   string
	Endpoint string // optional: override for MinIO / R2 / B2
	CDNBase  string // optional: if set, URL() returns CDN URL instead of presigning
}

// Store is the S3-backed MediaStore implementation.
type Store struct {
	client  *s3.Client
	presign *s3.PresignClient
	cfg     Config
}

// New creates an S3 Store.
//
// Endpoint override rationale (MinIO / Cloudflare R2 / Backblaze B2):
//   - BaseEndpoint sets the URL prefix for all S3 API calls.
//   - UsePathStyle = true is required for MinIO, which does not support
//     virtual-hosted-style URLs (bucket.endpoint). Cloudflare R2 works
//     with the default (virtual-hosted), but UsePathStyle is harmless.
//
// Credentials: loaded from the environment in the standard AWS credential
// chain (AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY, instance profile, etc.).
// This makes the implementation work unmodified on EC2/ECS/EKS and with
// MinIO's ACCESS_KEY / SECRET_KEY env vars.
func New(cfg Config) (*Store, error) {
	opts := []func(*awsconfig.LoadOptions) error{
		awsconfig.WithRegion(cfg.Region),
	}

	awsCfg, err := awsconfig.LoadDefaultConfig(context.Background(), opts...)
	if err != nil {
		return nil, fmt.Errorf("media/s3: load AWS config: %w", err)
	}

	s3Opts := []func(*s3.Options){}
	if cfg.Endpoint != "" {
		s3Opts = append(s3Opts, func(o *s3.Options) {
			o.BaseEndpoint = aws.String(cfg.Endpoint)
			o.UsePathStyle = true
		})
	}

	client := s3.NewFromConfig(awsCfg, s3Opts...)
	presign := s3.NewPresignClient(client)

	return &Store{
		client:  client,
		presign: presign,
		cfg:     cfg,
	}, nil
}

// Put uploads content to S3 using PutObject.
// The content is streamed from r — the AWS SDK does not buffer the body
// unless ContentLength is unknown, in which case it buffers to determine size.
// Since we always have the full bytes in memory (see MediaService.Upload),
// we set ContentLength explicitly to avoid SDK-level buffering.
func (s *Store) Put(ctx context.Context, key string, r io.Reader, contentType string) error {
	// io.Reader from bytes.NewReader always supports length via type assertion.
	// For streaming readers without a known length, AWS SDK buffers internally.
	_, err := s.client.PutObject(ctx, &s3.PutObjectInput{
		Bucket:      aws.String(s.cfg.Bucket),
		Key:         aws.String(key),
		Body:        r,
		ContentType: aws.String(contentType),
	})
	if err != nil {
		return fmt.Errorf("media/s3: put %q: %w", key, err)
	}
	return nil
}

// Get downloads the object at key from S3, returning its body as a streaming
// ReadCloser. The caller must close the returned reader.
// Returns media.ErrNotFound for NoSuchKey errors.
func (s *Store) Get(ctx context.Context, key string) (io.ReadCloser, error) {
	out, err := s.client.GetObject(ctx, &s3.GetObjectInput{
		Bucket: aws.String(s.cfg.Bucket),
		Key:    aws.String(key),
	})
	if err != nil {
		if isNotFound(err) {
			return nil, media.ErrNotFound
		}
		return nil, fmt.Errorf("media/s3: get %q: %w", key, err)
	}
	return out.Body, nil
}

// Delete removes the object at key from S3.
// S3 DeleteObject is idempotent — deleting a non-existent key is not an error.
func (s *Store) Delete(ctx context.Context, key string) error {
	_, err := s.client.DeleteObject(ctx, &s3.DeleteObjectInput{
		Bucket: aws.String(s.cfg.Bucket),
		Key:    aws.String(key),
	})
	if err != nil {
		return fmt.Errorf("media/s3: delete %q: %w", key, err)
	}
	return nil
}

// URL returns a public URL for the object.
//
// Two modes:
//   - CDN base set: returns "{CDNBase}/{key}" — no signing, no expiry.
//     Assumes the CDN has read access to the bucket and handles caching.
//   - No CDN base: returns a presigned GetObject URL with a 1-hour expiry.
//     Suitable for private buckets where all access is through the app.
//
// Note: presigned URLs are generated on every call. For hot paths (timeline
// rendering), the calling code should cache the URL for a duration shorter
// than the presign expiry.
func (s *Store) URL(ctx context.Context, key string) (string, error) {
	if s.cfg.CDNBase != "" {
		return strings.TrimRight(s.cfg.CDNBase, "/") + "/" + key, nil
	}

	req, err := s.presign.PresignGetObject(ctx, &s3.GetObjectInput{
		Bucket: aws.String(s.cfg.Bucket),
		Key:    aws.String(key),
	}, s3.WithPresignExpires(time.Hour))
	if err != nil {
		return "", fmt.Errorf("media/s3: presign %q: %w", key, err)
	}
	return req.URL, nil
}

// isNotFound returns true for S3 NoSuchKey errors.
func isNotFound(err error) bool {
	var nsk *s3types.NoSuchKey
	var nob *s3types.NotFound
	return errors.As(err, &nsk) || errors.As(err, &nob)
}
```

```go
// MediaAttachmentMeta is stored as JSONB in media_attachments.meta.
// Phase 1 populates Original for images; Small is populated in Phase 2
// when thumbnail generation is implemented.
type MediaAttachmentMeta struct {
	Original *MediaDimensions `json:"original,omitempty"`
	Small    *MediaDimensions `json:"small,omitempty"` // Phase 2: thumbnail
	Focus    *FocalPoint      `json:"focus,omitempty"` // user-supplied focal point
}

// MediaDimensions describes the spatial/temporal properties of a media file.
// Used for both image and video; video-only fields are zero for images.
type MediaDimensions struct {
	Width     int     `json:"width"`
	Height    int     `json:"height"`
	Size      string  `json:"size"`             // "1920x1080"
	Aspect    float64 `json:"aspect"`           // width / height
	Duration  float64 `json:"duration,omitempty"` // seconds; video only
	FrameRate string  `json:"frame_rate,omitempty"` // "30/1"; video only
	Bitrate   int     `json:"bitrate,omitempty"`    // bits/s; video only
}

// FocalPoint is the user-supplied focus point for image cropping.
// Coordinates are in the range [-1.0, 1.0]; (0, 0) is the centre.
type FocalPoint struct {
	X float64 `json:"x"`
	Y float64 `json:"y"`
}

// UploadParams carries the parsed multipart upload request.
type UploadParams struct {
	Data        []byte   // entire file contents (already validated ≤ MaxBytes)
	ContentType string   // detected MIME type
	Description string   // alt text (may be empty)
	FocalPoint  *FocalPoint
}

// like Ivory, Tusky, and Elk render attachments correctly.

	ID          string               `json:"id"`
	Type        string               `json:"type"`         // "image"|"gifv"|"video"|"audio"|"unknown"
	URL         string               `json:"url"`
	PreviewURL  *string              `json:"preview_url"`  // null until thumbnail generated
	RemoteURL   *string              `json:"remote_url"`   // null for local uploads
	Description *string              `json:"description"`  // null if no alt text
	Blurhash    *string              `json:"blurhash"`     // null for non-images
```

### `MediaService`

```go
// MediaService orchestrates media upload, validation, storage, and retrieval.
type MediaService struct {
	store    store.Store       // DB access (AttachmentStore)
	media    media.MediaStore  // pluggable storage backend
	cache    cache.Store       // for presigned URL caching (see URL helper below)
	maxBytes int64
}

// NewMediaService constructs a MediaService.
func NewMediaService(s store.Store, m media.MediaStore, c cache.Store, maxBytes int64) *MediaService {
	return &MediaService{store: s, media: m, cache: c, maxBytes: maxBytes}
}

// Upload validates, stores, and records a new media attachment.
//
// Upload pipeline:
//  1. Size is enforced by the handler via io.LimitReader before calling Upload;
//     params.Data already contains the bounded bytes.
//  2. Content type is detected from the first 512 bytes using http.DetectContentType.
//  3. The detected type is validated against the allowlist.
//  4. For images: the image is decoded to extract dimensions and compute a blurhash.
//  5. A storage key is generated and the bytes are PUT to the MediaStore.
//  6. A media_attachments row is inserted in the database.
//  7. If the DB insert fails, a best-effort Delete is issued to the storage backend
//     to avoid orphaned objects (see Partial Upload Rollback note below).
//
// Partial upload rollback note:
//   The Put→Insert sequence is NOT atomic. If Insert fails after a successful Put,
//   a Delete is attempted. If Delete also fails, the object becomes orphaned in
//   storage. An admin cleanup job (Phase 3) can sweep for orphaned objects by
//   comparing storage keys against media_attachments rows.
func (s *MediaService) Upload(ctx context.Context, accountID string, params UploadParams) (*MediaAttachmentResponse, error) {
	// Detect and validate content type.
	detectedType := http.DetectContentType(params.Data[:min(512, len(params.Data))])
	// http.DetectContentType may return "image/jpeg; charset=utf-8" — strip parameters.
	mediaType, _, _ := mime.ParseMediaType(detectedType)

	mastodonType, allowed := media.AllowedContentTypes[mediaType]
	if !allowed {
		return nil, &ValidationError{Message: fmt.Sprintf("unsupported media type: %s", mediaType)}
	}

	// Generate storage key.
	key := media.StorageKey(mediaType)

	// For images: extract dimensions and compute blurhash.
	var meta *MediaAttachmentMeta
	var blurhashVal *string

	if mastodonType == "image" || mastodonType == "gifv" {
		m, bh, err := processImage(params.Data)
		if err != nil {
			return nil, &ValidationError{Message: "could not process image: " + err.Error()}
		}
		meta = &MediaAttachmentMeta{Original: m}
		if bh != "" {
			blurhashVal = &bh
		}
	}

	// Store the file.
	if err := s.media.Put(ctx, key, bytes.NewReader(params.Data), mediaType); err != nil {
		return nil, fmt.Errorf("media upload: store: %w", err)
	}

	// Build the public URL (cached for presigned URL efficiency).
	publicURL, err := s.urlForKey(ctx, key)
	if err != nil {
		// Best-effort cleanup — don't surface the delete error, just log.
		_ = s.media.Delete(ctx, key)
		return nil, fmt.Errorf("media upload: get URL: %w", err)
	}

	// Encode meta as JSONB.
	var metaJSON json.RawMessage
	if meta != nil {
		metaJSON, _ = json.Marshal(meta)
	}

	// Insert DB row.
	var descPtr *string
	if params.Description != "" {
		descPtr = &params.Description
	}
	attachment, err := s.store.CreateMediaAttachment(ctx, db.CreateMediaAttachmentParams{
		ID:          uid.New(),
		AccountID:   accountID,
		Type:        mastodonType,
		StorageKey:  key,
		Url:         publicURL,
		Description: descPtr,
		Blurhash:    blurhashVal,
		Meta:        metaJSON,
	})
	if err != nil {
		_ = s.media.Delete(ctx, key) // best-effort rollback
		return nil, fmt.Errorf("media upload: insert row: %w", err)
	}

	return toMediaResponse(attachment), nil
}

// Get returns the media attachment with the given ID.
// Returns domain.ErrNotFound if no attachment exists with that ID.
// The URL in the response is resolved through urlForKey so that presigned
// URLs are served from cache rather than re-signed on every poll.
func (s *MediaService) Get(ctx context.Context, id string) (*MediaAttachmentResponse, error) {
	attachment, err := s.store.GetMediaAttachment(ctx, id)
	if err != nil {
		return nil, err
	}
	resp := toMediaResponse(attachment)
	if url, err := s.urlForKey(ctx, attachment.StorageKey); err == nil {
		resp.URL = url
	}
	return resp, nil
}

// UpdateDescription sets the alt text on an existing attachment.
// Only the owning account may update the description.
func (s *MediaService) UpdateDescription(ctx context.Context, id, accountID, description string) (*MediaAttachmentResponse, error) {
	attachment, err := s.store.GetMediaAttachment(ctx, id)
	if err != nil {
		return nil, err
	}
	if attachment.AccountID != accountID {
		return nil, domain.ErrForbidden
	}

	var focalPoint *json.RawMessage // TODO: accept focal_point param from handler
	updated, err := s.store.UpdateMediaAttachment(ctx, db.UpdateMediaAttachmentParams{
		ID:          id,
		AccountID:   accountID,
		Description: &description,
		Meta:        focalPoint,
	})
	if err != nil {
		return nil, fmt.Errorf("media update: %w", err)
	}
	return toMediaResponse(updated), nil
}

// urlForKey returns the public URL for a storage key, with caching for presigned URLs.
//
// When the S3 driver is used without a CDN, MediaStore.URL() issues an AWS SDK call
// to generate a presigned GetObject URL on every invocation. To avoid this overhead
// during timeline rendering (which calls urlForKey for every attachment), we cache
// the URL for 50 minutes — safely under the 1-hour presign expiry.
//
// When the local driver or S3+CDN is used, MediaStore.URL() is a pure string
// concatenation (no network call), so the cache lookup adds negligible overhead
// but the result is stored anyway for consistency.
//
// Cache key: "mediaurl:{storageKey}"
// TTL: 50 minutes (10-minute buffer before presigned URL expires)
func (s *MediaService) urlForKey(ctx context.Context, storageKey string) (string, error) {
	const ttl = 50 * time.Minute
	cacheKey := "mediaurl:" + storageKey

	var cached string
	if hit, _ := cache.GetJSON(ctx, s.cache, cacheKey, &cached); hit {
		return cached, nil
	}

	url, err := s.media.URL(ctx, storageKey)
	if err != nil {
		return "", err
	}
	_ = cache.SetJSON(ctx, s.cache, cacheKey, url, ttl)
	return url, nil
}

// processImage decodes the image data, extracts dimensions, and computes a blurhash.
// Returns (MediaDimensions, blurhash string, error).
// Blurhash components: 4 horizontal × 3 vertical (Mastodon's default).
func processImage(data []byte) (*MediaDimensions, string, error) {
	img, _, err := image.Decode(bytes.NewReader(data))
	if err != nil {
		return nil, "", fmt.Errorf("decode image: %w", err)
	}
	bounds := img.Bounds()
	w := bounds.Dx()
	h := bounds.Dy()

	var aspect float64
	if h > 0 {
		aspect = float64(w) / float64(h)
	}

	dims := &MediaDimensions{
		Width:  w,
		Height: h,
		Size:   fmt.Sprintf("%dx%d", w, h),
		Aspect: aspect,
	}

	bh, err := blurhash.Encode(4, 3, img)
	if err != nil {
		// Blurhash failure is non-fatal — return dims without blurhash.
		return dims, "", nil
	}
	return dims, bh, nil
}

// toMediaResponse maps a store.MediaAttachment to the Mastodon API response shape.
func toMediaResponse(a store.MediaAttachment) *MediaAttachmentResponse {
	resp := &MediaAttachmentResponse{
		ID:   a.ID,
		Type: a.Type,
		URL:  a.Url,
	}
	if a.Description != nil {
		resp.Description = a.Description
	}
	if a.Blurhash != nil {
		resp.Blurhash = a.Blurhash
	}
	if a.PreviewUrl != nil {
		resp.PreviewURL = a.PreviewUrl
	}
	if a.RemoteUrl != nil {
		resp.RemoteURL = a.RemoteUrl
	}
	if a.Meta != nil {
		var m MediaAttachmentMeta
		if err := json.Unmarshal(a.Meta, &m); err == nil {
			resp.Meta = &m
		}
	}
	return resp
}
```

## 5. `internal/api/mastodon/media.go`

```go
package mastodon

import (
	"errors"
	"io"
	"net/http"

	"github.com/go-chi/chi/v5"

	"github.com/yourorg/monstera-fed/internal/api"
	"github.com/yourorg/monstera-fed/internal/middleware"
	"github.com/yourorg/monstera-fed/internal/service"
)

// MediaHandler holds dependencies for the media endpoints.
type MediaHandler struct {
	media    *service.MediaService
	logger   *slog.Logger
	maxBytes int64
}

// POST /api/v2/media
//
// Accepts a multipart/form-data body with:
//   - file:        the media file (required)
//   - description: alt text (optional)
//   - focus:       focal point "x,y" (optional, e.g. "0.5,-0.3")
//
// Flow:
//  1. Authenticate (RequireAuth middleware).
//  2. Parse multipart form with a memory limit to avoid large files in memory;
//     the limit is the same as MaxBytes — the file will be buffered once in Upload.
//  3. Read the file body through an io.LimitReader to enforce the size cap before
//     passing to the service (returns 422 if limit is exceeded mid-read).
//  4. Call MediaService.Upload.
//  5. Return 202 Accepted with the MediaAttachment JSON (async-processing convention;
//     Mastodon clients poll GET /api/v1/media/:id until url is non-null).
func (h *MediaHandler) Upload(w http.ResponseWriter, r *http.Request) {
	account := middleware.AccountFromContext(r.Context())

	if err := r.ParseMultipartForm(h.maxBytes); err != nil {
		api.WriteError(w, http.StatusUnprocessableEntity, "could not parse form")
		return
	}

	file, _, err := r.FormFile("file")
	if err != nil {
		api.WriteError(w, http.StatusUnprocessableEntity, "file field is required")
		return
	}
	defer file.Close()

	// Read the entire file, capped at maxBytes+1 to detect oversized uploads.
	lr := io.LimitReader(file, h.maxBytes+1)
	data, err := io.ReadAll(lr)
	if err != nil {
		api.WriteError(w, http.StatusUnprocessableEntity, "could not read file")
		return
	}
	if int64(len(data)) > h.maxBytes {
		api.WriteError(w, http.StatusUnprocessableEntity,
			fmt.Sprintf("file exceeds maximum size of %d bytes", h.maxBytes))
		return
	}

	params := service.UploadParams{
		Data:        data,
		Description: r.FormValue("description"),
	}
	if fp := r.FormValue("focus"); fp != "" {
		params.FocalPoint = parseFocalPoint(fp)
	}

	attachment, err := h.media.Upload(r.Context(), account.ID, params)
	if err != nil {
		var ve *service.ValidationError
		if errors.As(err, &ve) {
			api.WriteError(w, http.StatusUnprocessableEntity, ve.Message)
			return
		}
		api.WriteAppError(w, h.logger, &api.AppError{
			Status:  http.StatusInternalServerError,
			Message: "internal server error",
			Err:     err,
		})
		return
	}

	// 202 Accepted — Mastodon clients poll until url is populated.
	api.WriteJSON(w, http.StatusAccepted, attachment)
}

// GET /api/v1/media/:id
//
// Returns the current state of a media attachment. Mastodon clients call this
// to poll for async processing completion (url becomes non-null when ready).
// In Phase 1, processing is synchronous so url is always set on first response.
func (h *MediaHandler) Get(w http.ResponseWriter, r *http.Request) {
	id := chi.URLParam(r, "id")
	attachment, err := h.media.Get(r.Context(), id)
	if err != nil {
		if errors.Is(err, domain.ErrNotFound) {
			api.WriteError(w, http.StatusNotFound, "Record not found")
			return
		}
		api.WriteAppError(w, h.logger, &api.AppError{
			Status: http.StatusInternalServerError, Message: "internal server error", Err: err,
		})
		return
	}
	api.WriteJSON(w, http.StatusOK, attachment)
}

// PUT /api/v1/media/:id
//
// Updates the alt text (description) and optionally the focal point of an
// unattached media attachment. Returns 422 if the attachment is already
// attached to a status (Mastodon spec: cannot update attached media).
func (h *MediaHandler) UpdateDescription(w http.ResponseWriter, r *http.Request) {
	account := middleware.AccountFromContext(r.Context())
	id := chi.URLParam(r, "id")

	if err := r.ParseForm(); err != nil {
		api.WriteError(w, http.StatusUnprocessableEntity, "invalid form data")
		return
	}
	description := r.FormValue("description")

	attachment, err := h.media.UpdateDescription(r.Context(), id, account.ID, description)
	if err != nil {
		switch {
		case errors.Is(err, domain.ErrNotFound):
			api.WriteError(w, http.StatusNotFound, "Record not found")
		case errors.Is(err, domain.ErrForbidden):
			api.WriteError(w, http.StatusForbidden, "This action is not allowed")
		default:
			api.WriteAppError(w, h.logger, &api.AppError{
				Status: http.StatusInternalServerError, Message: "internal server error", Err: err,
			})
		}
		return
	}
	api.WriteJSON(w, http.StatusOK, attachment)
}

// parseFocalPoint parses a Mastodon focal point string "x,y" into a FocalPoint.
// Returns nil on parse failure (focal point is optional).
func parseFocalPoint(s string) *service.FocalPoint {
	var x, y float64
	if _, err := fmt.Sscanf(s, "%f,%f", &x, &y); err != nil {
		return nil
	}
	// Clamp to valid range.
	x = max(-1.0, min(1.0, x))
	y = max(-1.0, min(1.0, y))
	return &service.FocalPoint{X: x, Y: y}
}
```

## 6. Phase 2 Async Processing Note

**Where the job would be enqueued:** At the end of `MediaService.Upload`, after the DB row is inserted, a NATS JetStream message would be published to a new `MEDIA_PROCESS` stream:

```json
{
  "media_attachment_id": "01HQ3M...",
  "storage_key": "media/2026/02/24/V1StGXR8.jpg",
  "content_type": "image/jpeg",
  "account_id": "01HQ1A..."
}
```

**What a Phase 2 worker would produce:**
1. Fetch the original from storage.
2. Resize to a thumbnail (e.g. 400px wide, JPEG, quality 80) — store under a sibling key (`media/2026/02/24/V1StGXR8_small.jpg`).
3. For video: extract the first keyframe as an image; generate thumbnail and blurhash from it.
4. For video: extract duration, frame rate, and bitrate via `ffprobe`.
5. Update the `media_attachments` row: set `preview_url`, `blurhash`, and the full `meta` JSON.
6. Push an SSE `media_attachment_update` event to the owning account (Mastodon clients display the preview once it appears).

**Phase 1 simplification:** Blurhash and dimensions are computed synchronously during upload for images. `preview_url` is null until Phase 2. Mastodon clients handle null `preview_url` gracefully — they show the full image without a low-res placeholder.
